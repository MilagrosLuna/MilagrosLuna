## Projects
### Vehicle Fleet Management
This project is a vehicle fleet management system designed to simplify the administration of cars, services, and rentals. It allows users to load and manage detailed information for each vehicle, record services, manage rentals, and view specific sections for each category. Additionally, it includes a details section that displays consolidated information by license plate and two general charts that provide a view of the economic balance and the number of services per license plate.

### Job Management
Welcome to the job management section. Here, you can efficiently load, view, and modify jobs. This tool is designed to assist administrators in keeping track of incoming work, providing a clear and up-to-date view of all ongoing tasks.
